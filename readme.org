* To build
  #+begin_src sh
    bb generate
  #+end_src

  Produces =docs/index.html=, =docs/simple.html=, and =docs/resume.txt=.

* To build and deploy
  #+begin_src sh
    ./deploy.sh
  #+end_src

* For development
  Start a REPL and connect:
  #+begin_src sh
    bb nrepl-server
  #+end_src

  Then from your editor:
  #+begin_src clojure
    (require '[resume.core :as core])
    (require '[resume.server :as server])

    ;; Generate all files
    (core/generate)

    ;; Or start a dev server at localhost:3000
    (server/start!)

    ;; Regenerate and restart
    (server/restart!)
  #+end_src

  Or run the server directly:
  #+begin_src sh
    bb serve
  #+end_src
